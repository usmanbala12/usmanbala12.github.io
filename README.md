## Setup

To run this project locally:

1. Clone the repository
   ```
   git clone https://github.com/usmanbala12/usmanbala12.github.io.git
   ```
2. Navigate to the project directory
   ```
   cd usmanbala12.github.io
   ```
3. Run the command
   ```
   bundle exec jekyll serve
